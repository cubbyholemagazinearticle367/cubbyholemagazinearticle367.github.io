# Projects

## KnowledgeOps Copilot

A local Python desktop app for searching documents and returning source-backed answers.

![KnowledgeOps Copilot app screenshot](/assets/images/knowledgeops-demo.png)

### Why it matters

Many teams have policies, SOPs, onboarding notes, PDFs, and how-to documents scattered across different folders. KnowledgeOps Copilot explores how a lightweight local tool can make that information easier to search while keeping answers tied to source files.

### Built with

- Python
- Tkinter GUI
- Local file ingestion
- Search scoring
- Source-backed answer display
- PowerShell setup/build scripts
- PyInstaller EXE packaging
