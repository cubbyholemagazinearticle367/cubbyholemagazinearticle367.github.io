# Pragmatic-style dark layout patch

This patch updates the GitHub Pages site to use a dark editorial blog layout with:

- Large title banner at the top
- Assorted navigation links under the title
- Main content column
- Right sidebar with "By Gabriel Pina" and links
- Dark mode by default
- Screenshot path set to `/assets/images/knowledgeops-demo.png`

The design is inspired by simple engineering-blog layouts, but the stylesheet and branding are original to Pina Applied AI Lab.
